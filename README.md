# Merchify

Merchify is an AI-powered fashion e-commerce platform where users can explore trending clothing, customize designs, and purchase fashion products online.

## Features (Planned)
- Location-based fashion drops
- 3D T-shirt customization
- Creator marketplace for uploading designs
- Design battle with community voting
- AI outfit builder
- Subscription benefits for premium users
- Reward coupons after orders
- Chatbot for customer support

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js (backend)
- MongoDB (database)

## Project Status
This project is currently under development. The basic HTML structure including homepage, login page, and register page has been created. More features and pages will be added in upcoming updates.
